Welcome to AFCDTech, Andrew Fan/Sparen's Github Pages Site!

Welcome! This site was made in order to showcase and advertise projects that I, Andrew Fan (known as Sparen elsewhere on the internet) have made, as well as promote knowledge and usage of the scripting engine and associated language of Touhou Danmakufu ph3. Here you will find my projects, my services for the Danmakufu community, and my tutorials for Touhou Danmakufu ph3.

My primary services for the Touhou Danmakufu community are my script video series DNHArt/DNHLP, the Contest Database and related data visualization tools, and the Danmakufu Video Database. These can be accessed via the sidebar.

If you're looking for my Danmakufu projects, my official Project Release Thread on Maidens of the Kaleidoscope can be accessed at: Sparen's Code Dump. The Projects link on the sidebar links to an overview of a number of my projects, including my Open Source projects hosted on Github.
