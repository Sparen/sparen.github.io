Welcome to AFCDTech, Andrew Fan/Sparen's Github Pages Site!

Welcome! This site was made in order to showcase and advertise projects that I, Andrew Fan (known as Sparen elsewhere on the internet) have made, as well as promote knowledge and usage of the scripting engine and associated language of Touhou Danmakufu ph3. Here you will find my projects, my Youtube and Advertisement services for the Danmakufu community, and my tutorials for Touhou Danmakufu ph3.

if You're looking for my Danmakufu projects, my official Project Release Thread on Maidens of the Kaleidoscope can be accessed at: Sparen's Code Dump. The Projects link on the sidebar links to an overview of a number of my projects, including my Open Source projects

What to advertise your Touhou website/blog/etc. on this website free of charge? Visit my Advertisements page for more details!

P.S. This site is NOT DESIGNED FOR MOBILE. I use preformatted code blocks and embedded youtube videos, which are not supported by browsers (code blocks completely mess up everything in mobile and iOS does not support embedded Youtube videos because of Flash).
