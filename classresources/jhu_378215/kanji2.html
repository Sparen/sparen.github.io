This is my primer for those entering Nakao-sensei's class from Katagiri-sensei's class. This page specifically covers the non-shaded kanji phrases located in Genki I. Here you will find a number of questions in quiz format. Please select the correct answer, and if prompted to type it, please type the response.

If you find errors, please inform me via text, Facebook, or e-mail. If you use this website's comment form, I will discover your comment a month after the class has ended. I guarantee it.
