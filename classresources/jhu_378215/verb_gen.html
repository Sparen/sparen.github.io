Verb Particles

Here you will find automatically generated questions in quiz format. Please select the correct particle.

You may want to consider zooming in if you cannot read the furigana/ruby on top of the kanji. If you see words you are not familiar with, please use Google Translate or some other software - limiting direct objects based off of the lesson of the verb makes writing the software significantly more complicated than it needs to be for basic functionality.

If you find errors, a grammatically/situationally incorrect sentence, or a sentence that is downright insulting, please inform me via text, Facebook, or e-mail. The random nature of the program means that strange and unusual combinations of words may appear. Chances are that I will have forgotten a scenario in which case a certain particle is acceptable/not acceptable. Additionally, some verbs do not take direct objects, so particles have been set despite them not always being required.

This program is Open Source but it is highly recommended that you not copy or reuse the code here, as it was written in an extremely haphazard way.
