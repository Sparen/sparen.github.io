Danmakanvas - Stationary Astroid (Parametric Graph)

Danmakanvas is a Javascript Danmaku simulation made by Sparen. It does NOT work the same way as Danmakufu. Please be advised that the speed at which the simulation runs is therefore not equivalent to the speed that the code would run in Danmakufu.

    let angleT = 0;
    loop(30){
        //NOTE: The bullet color does not correspond to the graphic used in the code below
        CreateShotA1(ObjMove_GetX(objBoss) + 120*cos(angleT) ^ 3, ObjMove_GetY(objBoss) + 120*sin(angleT) ^ 3, 0, angleT, 0, 0);
        angleT += 360/30;
    }
