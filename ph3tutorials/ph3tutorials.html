PH3 Tutorials

Welcome to my Danmakufu Tutorials for Touhou Danmakufu ph3!

For future reference, please avoid linking to any of the tutorials in this page. I suggest that you link to this page instead, as if I get a new domain, this page will be the one with a link to the domain name, since Github Pages isn't really meant for large scale things like this.

These tutorials use Javascript, so I suggest that you turn it on for this site, as I have included comprehension questions that give feedback so that those using these tutorials will have a better understanding of why things work the way they do.

The Danmakufu Wiki and the Ph3 Function list will be linked on every tutorial page, and I also hope to integrate existing tutorials into the framework of this tutorial.

I hope you find this helpful!
