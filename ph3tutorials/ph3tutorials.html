My Tutorials and Guides for Danmakufu ph3

Welcome to my Danmakufu Tutorials for Touhou Danmakufu ph3! Unless otherwise stated, all tutorials are written by me, Sparen.

Quick Link to Compilation of Danmaku Uploaders on Youtube

For future reference, please avoid linking to any of the tutorials in this page, with the exception of the ones in the extra unit. I suggest that you link to this page instead, as if I get a new domain, this page will be the one with a link to the domain name, since Github Pages isn't really meant for large scale things like this.

Please note that these are not so much tutorials as they are step by step guides to learn how to do something. Therefore, there will be a lot of detail, and these are not meant for you to make a script in a day - take your time and learn the concepts.

These Danmakufu tutorials use Javascript, so I suggest that you turn it on for this site, as I have included comprehension questions that give feedback so that those using these tutorials will have a better understanding of why things work the way they do.

The Danmakufu Wiki and the ph3 Function list will be linked on every tutorial page, and I also hope to integrate existing tutorials into the framework of this tutorial.

All tutorials undergo peer-review. However, newer tutorials (especially brand new tutorials) will post likely have typos and errors. If you find incorrect/confusing information or typos, feel free to PM me on either Skype (sparenofiria) or Maidens of the Kaleidoscope, or fill out the comment form below (Note that I am not informed of new comments and that I need to manually check them). If contacting by skype, you MUST state that you are asking a question about Danmakufu or I may block you.

If you would like to leave a comment, please fill out this form.

I hope you find this helpful!
